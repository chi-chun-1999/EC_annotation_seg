介紹

自動計算 Key points

然而為了標記的方便性，程式的預設上是標記完左心室與左心房後，會自動計算 Key points 的位置，這也是為什麼前面提到必須要標記左心房與左心室的原因。

而自動計算 Key points 的方法主要是計算左心室與左心房polygons上每一點彼此之間的距離，當距離小於 2 時，將左心房上的點視為 key points 的候選點（下圖中紅色的點即為候選點）。

計算完後進一步計算 x 軸最左與最右的候選點，將其視為 key point1 與 key point2。

P.S. 然而在一些情況下，特別是手動標記左心房與左心室時，往往無法透過自動計算找到 Key points，而這時如果再使用 EqualLAM 與 Snake LAM，會跳出錯誤訊息提醒使用者無法找到 Key points，這個時候可能就必須修改左心房與左心室的標記點，不然就是手動標記 Key points。


手動標記 Key points

手動標記的功能，主要解決了前述提到無法透過自動計算找到 Key points 的問題，同時也針對自動計算找到 Key points 不夠準確的問題提供了解決方案，進而有效地改善對左心房心肌的標記效果。下圖中說明因為自動計算找到 Key points 不夠準確，導致左心房心肌標記了額外的部分。


手動標記 Key points 的操作如下：

  1. 在左邊工具列中選擇 Add Key point。



  2. 接著按下滑鼠左鍵標記 Key point 1 與 Key point 2 ，而程式的設定為第一個點為 Key point 1 （紅色標記點），第二個點為 Key point 2 （綠色標記點）。



  3. 透過手動標記 Key points，最後再應用 EqualLAM 後，明顯改善了對左心房心肌的標記，與第一步驟相比取得了顯著改善。
